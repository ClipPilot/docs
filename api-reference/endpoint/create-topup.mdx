---
title: "Create Top-Up"
api: "POST https://api.cuttr.com/api/platform/v1/wallet/topup"
description: "Create a Stripe payment intent to add funds to your wallet"
---

Creates a Stripe payment intent to top up your API wallet. Minimum amount is **$5.00** (500 cents). Use the returned `clientSecret` to complete the payment with Stripe.

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key.
</ParamField>

### Body parameters

<ParamField body="amount_cents" type="integer" required>
  Amount to add in cents. Minimum `500` ($5.00).
</ParamField>

### Response

<ResponseField name="clientSecret" type="string">
  Stripe client secret to complete the payment on the frontend.
</ResponseField>

<ResponseField name="paymentIntentId" type="string">
  Stripe payment intent ID. Use this to check the top-up status.
</ResponseField>

<ResponseExample>
```json 200
{
  "clientSecret": "pi_xxx_secret_xxx",
  "paymentIntentId": "pi_xxx"
}
```

```json 400
{
  "error": "Minimum top-up is $5.00 (500 cents)"
}
```
</ResponseExample>
