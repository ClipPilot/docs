---
title: "Get Top-Up Status"
api: "GET https://api.cuttr.com/api/platform/v1/wallet/topup/{paymentIntentId}/status"
description: "Check whether a top-up has been credited to your wallet"
---

Check if a top-up payment has been processed and credited to your wallet balance.

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key.
</ParamField>

### Path parameters

<ParamField path="paymentIntentId" type="string" required>
  The Stripe payment intent ID returned from the [Create Top-Up](/api-reference/endpoint/create-topup) endpoint.
</ParamField>

### Response

<ResponseField name="credited" type="boolean">
  `true` if the payment has been processed and credited to your wallet.
</ResponseField>

<ResponseExample>
```json 200
{
  "credited": true
}
```
</ResponseExample>
